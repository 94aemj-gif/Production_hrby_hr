Responsive — 5 direcciones

Misma capa visual (Ocean Teal, mismas cards, mismo botón Capturar). Solo cambia la estrategia del header y cómo se adapta a tablet/mobile. Abre cada uno, ve las 3 vistas apiladas (desktop · tablet · mobile), y dime la letra.

Estandarización aplicada a las 5

  • Breakpoints · 480 / 768 / 1024 / 1280
  • Container max-width · 1320 px
  • Type scale · clamp() en title + counter + metric
  • Tap targets · ≥ 44×44px
  • Capture btn · full-width mobile, min(100%, 360px) desktop
  • Card padding · 20 / 24 / 28 px (mobile/tablet/desktop)
  • Card radius · 16 / 18 / 22 px
  • Spacing · escala 4/8/12/16/24/32/48/64
